Hosting Servers

to host servers on a VPS or your own pc

visit here

you need to have docker installed for this

you can use WSL if you are using Windows, or use the terminal if you're using linux

requesting a server

please friend me on discord to request: keith1robloc

please make sure to include the following:

1. are you hosting on radminVPN (if yes, please specify the address and password of it)

2. what would you want to set the description to

3. what version are you hosting

4. what would you want the server name to be

5. the server's shared IP

6. would you want to use your discord username for creditting, if not, please write down your username you want to use

7. what time will you open the server (can be 24/7, and please write your timezone if it's not 24/7)

8. what is your server's player limit
